Netmon Appliance
 

Netmon Appliance provides you with instant awareness of your organization's mission-critical IT infrastructure. Netmon Appliance allows you to detect and repair problems and mitigate future issues before they affect end-users and customers.
Monitoring for simple services as: Web Port, Website Content, Mail Server Port, TCP Port.
Monitor server performance across multiple OSes; Supports monitoring Windows, Linux, Solaris, HP UX, IBM AIX servers via SNMP.
Monitor Servers for CPU, Memory and Disk Utilization, Services, Windows Services, Processes, Custom scripts, URLs (HTTP⁄ HTTPs), Files⁄ Folders.
Alerting when critical infrastructure components fail and recover, providing administrators with notice of important events. Alerts can be delivered via email, SMS.
